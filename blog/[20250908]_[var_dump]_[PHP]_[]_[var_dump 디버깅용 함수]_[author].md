`var_dump();` 는 PHP에서 **변수에 들어 있는 값과 그 타입(type)을 자세하게 출력해 주는 디버깅용 함수**예요.

### 특징

* 변수의 **데이터 타입**과 **값**을 함께 보여줌
* 배열이나 객체처럼 복잡한 구조도 계층적으로 출력
* `echo`나 `print_r()`보다 더 상세하게 확인 가능

### 예시

```php
<?php
$a = "Hello";
$b = 123;
$c = [1, 2, 3];

var_dump($a);
var_dump($b);
var_dump($c);
?>
```

### 출력 결과

```
string(5) "Hello"
int(123)
array(3) {
  [0]=>
  int(1)
  [1]=>
  int(2)
  [2]=>
  int(3)
}
```

👉 이렇게 문자열의 길이(`string(5)`), 정수 타입(`int`), 배열의 크기와 내부 요소까지 다 보여줍니다.

원래는 디버깅용으로 많이 쓰이고, 실제 운영 환경에서는 보안상 노출을 막기 위해 잘 쓰지 않습니다.
